# Manuals QR

<img src="https://github.com/TomPrint/ManualsQR/blob/main/ManualsQR.jpg" width="200" alt="Manuals QR" />

**Manuals QR** is a fully responsive web application built with:  
![Django](https://img.shields.io/badge/Django-092E20?logo=django&logoColor=white)

The app provides **easy access to PDF manuals and embedded videos** via QR codes,  
and enables **targeted marketing communication** delivered directly to users after scanning a QR code.

## ðŸš€ Main Features

- **Two user roles** with different permission levels.
- **Order creation**, supporting:
  - up to **4 PDF manuals**
  - **1 embedded video**
  - access via a **unique QR code**
- **Marketing communication system** using sliding banners displayed after QR scan.
- **Email notifications** sent to predefined addresses when a new order is created.
- **Water usage calculation algorithm**, presented directly to the customer.
- **Order search functionality** using keywords.
- Application available in **Polish** ![Polish language](https://img.shields.io/badge/Language-Polish-red?logo=poland&logoColor=white)
