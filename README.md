# <img src="https://github.com/TomPrint/ManualsQR/blob/main/ManualsQR.jpg" width="160" height="auto" align="center"/>

is a DJANGO <img src="https://cdn.worldvectorlogo.com/logos/django.svg" alt="django" width="17" height="17" align="center"/> fully responsive application that allows easy access to **PDF manuals** and **embedded videos** via QR codes, as well as sending **targeted marketing messages** to users scanning the QR codes.

Application uses CSS framework **Bootstrap 5** <img src="https://upload.wikimedia.org/wikipedia/commons/b/b2/Bootstrap_logo.svg" alt="bootstrap" width="25" height="auto" align="center"/><br />
It also uses some dynamic content created with **HTMX** <img src="https://github.com/Klewiu/CutIt/blob/main/htmx_logo.webp" alt="htmx" width="100" height="auto" align="center"/> and **_hyperscript** <img src="https://camo.githubusercontent.com/b952f7af5fec7f541c8d95291a6df1f55c04914b558f324101ad06d94c35cdc2/68747470733a2f2f68797065727363726970742e6f72672f696d672f6c696768745f6c6f676f2e706e67" alt="hyperscript" width="100" height="auto" align="center"/>

---

## Main features:
1. Two types of users with different permission levels.
2. Order creation allows adding up to **4 PDF manuals** and **1 embedded video**, all accessible via a **unique QR code**.
3. Marketing banners displayed to users when they scan the QR code.
4. Email notifications sent to designated addresses upon order creation.
5. **Water usage calculation algorithm** displayed to the user.
6. Search functionality for orders using keywords.
7. The application is in **Polish** <img src="https://upload.wikimedia.org/wikipedia/commons/thumb/e/e9/Flag_of_Poland_%28normative%29.svg/360px-Flag_of_Poland_%28normative%29.svg.png" width="20" height="auto" align="center" />

---

## Reports:
1. Generate PDFs of manuals or order summaries.
2. Export lists of PDFs/videos to **Excel**.
3. View usage statistics and QR scan reports.
4. Charts showing material usage and order statistics using **Chart.js** <img src="https://www.chartjs.org/img/chartjs-logo.svg" alt="chartjs" width="25" height="auto" align="center"/>.
